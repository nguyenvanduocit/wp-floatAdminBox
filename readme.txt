This plugin help you to float the post box. The post box will auto fly to your viewport when you scroll the admin edit post page.

== Description ==

By default, when you post articles in article submission interface, sidebar on the right does not automatically move follow your viewport, thus after data entry is complete, you have to scroll to the top of the box to work with above box, so rather inconvenient. 

This plugin automatically scrolls according to the sidebar of your viewport.

== Installation ==

1. Upload `wp-floatAdminBox` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
